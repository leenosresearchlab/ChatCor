프로젝트 생성

프로젝트란 하나의 챗봇에 적용되는 데이터 세트를 의미합니다. 프로젝트를 생성하면 시스템에는 프로젝트의 이름과 함께 네 개의 새로운 폴더가 만들어 집니다.

  • intents : 인텐트 결과값
  • intents_work : 인텐트 편집 파일
  • entities : 엔트리 값들로 합쳐진 엔티티값
  • entries : 사용자가 임의로 올리는 용어
  • sentences : 사용자가 임의로 등록하는 문장

분석, 정제는 sentences 폴더에 저장된 문장을 중심으로 처리 됩니다.

Entity 편집은 entries 폴더에 올라온 파일 내에 나열된 데이터를 중심으로 entities에 값을 저장합니다.

프로젝트 생성 실패

프로젝트를 생성할 수 없습니다. 입력 내용 확인 후 다시 실행해 주시기 바랍니다.
